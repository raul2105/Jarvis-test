# Jarvis Test Service

A simple FastAPI service for testing OpenClaw agent configuration.

## Structure
- `app/`: Contains the web service
- `openclaw/`: OpenClaw configuration (skills, cron jobs)
- `memory/`: Health check logs (generated by OpenClaw)

## Setup
1. Install dependencies: `pip install -r app/requirements.txt`
2. Start the service: `uvicorn app.main:app --host 0.0.0.0 --port 8000`
3. OpenClaw will monitor the `/health` endpoint periodically

## Health Check
The service provides a `/health` endpoint that returns:
```json
{
  "status": "healthy",
  "timestamp": "2026-03-23T21:30:00Z",
  "service": "jarvis-test"
}
```
